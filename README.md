# Ordenamientos básicos en Python

## Descripción

En esta actividad practicarás la implementación de algoritmos básicos de ordenamiento usando Python.

El programa deberá trabajar con una lista de números y mostrar cómo se puede ordenar utilizando distintos métodos básicos, como:

* Bubble Sort
* Selection Sort
* Insertion Sort

La actividad se realizará en GitHub Codespaces, por lo que no necesitas instalar Python en tu computadora. El entorno ya cuenta con Python 3.12 instalado.

## Objetivo

El objetivo de esta actividad es que el alumno comprenda la lógica básica detrás de algunos algoritmos de ordenamiento.

Al finalizar, el alumno deberá ser capaz de:

1. Crear funciones en Python.
2. Manipular listas de números.
3. Implementar algoritmos básicos de ordenamiento.
4. Ejecutar un programa desde la terminal.
5. Comparar el resultado de distintos métodos de ordenamiento.

## Cómo abrir el Codespace

1. Entra al repositorio de la actividad en GitHub.
2. Haz clic en el botón verde **Code**.
3. Selecciona la pestaña **Codespaces**.
4. Haz clic en **Create codespace on main**.
5. Espera unos segundos mientras se prepara el entorno.
6. Cuando termine de cargar, se abrirá VS Code en el navegador.

## Cómo ejecutar

En el terminal integrado de VS Code, ejecuta el siguiente comando:

```bash
python main.py
```

El programa deberá mostrar una lista original y después la misma lista ordenada usando los algoritmos implementados.

Ejemplo de salida esperada:

```text
Lista original: [5, 3, 8, 1, 2]

Bubble Sort: [1, 2, 3, 5, 8]
Selection Sort: [1, 2, 3, 5, 8]
Insertion Sort: [1, 2, 3, 5, 8]
```

## Entregable

El alumno deberá entregar el enlace a su repositorio de GitHub con la actividad resuelta.

El repositorio debe incluir:

1. Un archivo `main.py` que ejecute el programa.
2. Una o más funciones para realizar los ordenamientos.
3. Comentarios breves que expliquen qué hace cada algoritmo.
4. Evidencia de ejecución, como una captura de pantalla o una copia de la salida del programa.

Ejemplo de estructura del repositorio:

```text
ordenamientos-basicos-python/
├── main.py
└── README.md
```
