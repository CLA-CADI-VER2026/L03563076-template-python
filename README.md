# Ordenamientos básicos en Python

## Descripción
En esta actividad practicarás el uso de listas, funciones y algoritmos básicos de ordenamiento en Python.

El programa debe mostrar cómo ordenar una lista de números usando al menos tres métodos:

- Bubble Sort
- Selection Sort
- Insertion Sort

También puedes comparar tus resultados con la función nativa `sorted()` de Python.

## Cómo abrir el entorno
1. Haz clic en el botón verde **Code** → pestaña **Codespaces**.
2. Selecciona **Create codespace on main**.
3. Espera ~30 segundos. VS Code abrirá en el navegador con Python listo.

## Cómo ejecutar tu programa
En el terminal integrado de VS Code:

python main.py
